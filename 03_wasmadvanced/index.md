# 画像の「明るさ」の基本概念

- 色によって感じ方が違う
- 人間の明るさの感じ方は線形でない

# ガンマ補正とは？

- 人間の明るさの感じ方は線形でない
- なので、強さによって自然に見えるよう、ある程度ディスプレイでの出し方の補正をかけている
- これがガンマ補正

# なのでグレースケールにするときも...

- 「色によって感じ方が違う」3色をまとめるのでガンマ補正を考慮しないといけない
  - モニタでは3色それぞれに曲線がある
